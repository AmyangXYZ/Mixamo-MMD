# Beside.love

Live iconic anime moments.

You're Denji in the pool with Reze. It's night. She's right there. Swim closer. Splash water. See what happens.

## What This Is

Not a chatbot. No text box.

Interactive 3D anime scenes. You're embodied in the moment—gesture, touch, move. Characters react with authored animations and voiced lines.

Galgame meets WebGPU.

## Design

- **Playable moments:** Specific scenes—the pool, the cafe. Bounded and intimate.
- **Gesture interaction:** Touch, poke, move. No typing.
- **Authored quality:** Real animations, real voice. Curated, not generated.

## Tech

WebGPU renderer, MMD/PMX models, VMD animation, browser-based.
